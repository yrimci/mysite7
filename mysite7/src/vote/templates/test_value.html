{# 파이썬 주석을 HTML코드에 넣을 때 사용하는 표기방식#} {# {{변수, 함수}} : 변수나 함수의 반환값을 HTML에 포함시킬 때 사용 #} {# {% 탬플릿 태그 %} : 특정 태그를 바탕으로 연산을 할 수 있는 표기방식#} {#view함수가 넘겨준 사전데이터의 Key : Value는 템플릿에서 Key는 변수로써 사용되고 변수를 호출하면 저장된 Value를 사용할 수 있음 #}

{{ b }}
